# Reference Sample Credits

The analysis data files (`.analysis.json`) in this directory were derived from
audio samples rendered from the **FluidR3_GM** General MIDI SoundFont.

Only mathematical analysis metadata (spectral statistics, MFCCs, rhythm
features, etc.) is distributed here -- no audio is included.
